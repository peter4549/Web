웹 페이지에 가장 많이 사용되는 Top 5의 태그들. => 'html', 'head', 'body', 'meta', 'title' Documents structure. 'title' tag는 책 표지같은 기능.

html 작성 시, 두 부분으로 구분할 수 있다. 실제 웹페이지의 내용이 포함되는 부분과 그 페이지를 부가적으로 설명하는 타이틀이나 디코딩 방식 등이 그에 해당한다. html document 내에서 그들을 구분한다.

=> 본문은 'body' tag, 부가적인 설명은 'head' tag로 묶는다. html의 모든 태그들은 'head' & 'body' tag에 소속되게 되는것이다. 이는 약속이며 일반적으로 따라야 한다. 앞의 두 태그는 상당히 고위의 태그라 할 수 있다. => "<meta charset = "utf-8">" 'head' tag내에 작성되는 앞의 코드는 html 파일을 읽는 방식을 알려주는 것이다. 이것이 정의되지 않는다면 웹브라우저는 스스로 read방식을 선정한다. 만약 그 방식이 한글을 처리할 수 있는 방식이라면 글자는 깨지지 않을 것이다.

앞의 두 고위 태그('head','body')를 묶는 단 하나의 최고위 태그가 있다. 'html' tag이다. 그리고 그 위에 작성된 문서가 'html' 문서임을 알리는 "<!doctype html>" 을 작성한다.
